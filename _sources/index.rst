==========================================
Teach Mobile CSP 
==========================================

.. raw:: html

	<p> Welcome to the Teaching Mobile CS Principles site.</p>
	
	<p> This site is the teacher component that accompanies the Mobile CSP course for students. The units parallel the student course, but provide lesson plans, reflections, pedagogy and other content to help support the teaching of Mobile Computer Science Principles.</p>
